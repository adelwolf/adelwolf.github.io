Documentation

Intro

Adelbot is a easy-to-use multi-purpose administration bot for Discord. This article is meant to help anyone who has questions about adelbot!

Basic info

Each action for adelbot starts with "adel:", also known as a prefix.

After each prefix comes the command. The command specifies what you want adelbot to do

After the command there may be another required input, also known as an argument. This gives the bot more information about the specified task.

Role Configuration

In order to make adelbot function correctly, you must adjust a few simple things in your server.

First you should make a new role with the name "Muted", make sure it's typed exactly like that!

After you make the role, go into permissions and uncheck "Send Messages" for that role. This will make them unable to talk in text chat! Now the mute command will work properly!

In order for your staff to be able to use commands, make a "Staff" role, make sure it's typed exactly like that! The people with that role will be able to use moderation commands!

[OPTIONAL] If you want a NSFW system, add an NSFW role. Then disallow access to your default role in an NSFW channel, and allow access for the NSFW role. Now the NSFW command is set up!

Commands

Now we'll go over each command, what it does, and how to use it!

adel:info - This command gives you general information about the bot!

adel:kick @user [reason] - This kicks a specified user from your server. @user is the user mentioned, and [reason] is the reason to kick them.

adel:ban @user [reason] - This bans a specified user from your server. @user is the user mentioned, and [reason] is the reason to ban them.

adel:mute @user - REQUIRES CONFIGURATION! This mutes a specified user. @user is the user mentioned.

adel:nsfw - REQUIRES CONFIGURATION! This applies an NSFW role to yourself.

adel:filter - Enables and disables the word filter on a server.

adel:getlastmsg @user - Gets a user's last sent message seen by the bot.

adel:server - Shows information about the current server.

adel:user @user - Shows information about a mentioned user, or yourself if no user is specified.

adel:purge [number] - Deletes a specified amount of messages from a channel, [number] being the amount of messages you want to delete
